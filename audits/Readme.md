# Contract Audits

The contracts in this repository were audited by Nethermind Security in October 2025, the final report can be found in [NM_0411_0693.pdf](./NM_0411_0693.pdf).
A follow-up audit in February 2026 was conducted to cover changes to the contracts since the first audit, the final report for this can be found in [NM_0411_0817.pdf](./NM_0411_0817.pdf).
